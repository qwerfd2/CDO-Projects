Hello fellow developer! Thank you for visiting my program!

As you may know, this app is quite big and the code is very messy, so I outlined every major functionality below.

If any of these interests you, please don't hesitate to create a User ID and leave me a comment!

---------------------------------------

    1) 4 (.5?) games

a) 1st game, Ultimate Click
- Four click spot with random position and size
- Down-to-millisecond timer
- Smart suggestions based on time, high score, and click count

b) 2nd game, Modified Blackjack
- Random number generation with modified rules for player and computer
- Win-lose reason determination
- Smart suggestions based on player’s hand

c) 3rd game, Enhanced Pong
- Ball collision
- Speed adjustment based on score, random modifier for velocity when hit coin
- keyboard and touch control

d) 4th game, Skillcheck
- In range calculation of object
- Difficulty adjustment – checker speed, region width
- keyboard and touch control

d) 4.5th game, Festival
- Date-time related events and collectables

    2) Progress system

a) XP system
- XP awarded for playing game, completing achievements, and beating high scores
- XP-unlockable content
- Global leaderboard & anti-cheat integration

b) Achievement system
- Two achievement (one for grind, one for skill) for each game
- Two achievement exclusively for logged-in player
- Prompt of completion in main menu
- Awards (trophy, SFX, XP)

c) Local high score
- Auto-recorded gaming high score
- Incentive to come back and perfect the high score (XP, global leaderboard)

d) Global high score
- Shows who has a high score and what the score is when a game end
- Reward for competitiveness (XP, global leaderboard)
- Taunt message for previous high score owner if beaten
- Global leaderboard integration

e) Global leaderboard
- Real-time global leaderboard for top-XP and top high score players
- Integration with anti-cheat and ban system

f) Detailed stats
- Detailed gameplay and account analytics for personal account
- Macro analytics for the game as a whole
- Detailed XP rules

    3) Cloud save system
a) User ID
- Saves a player’s setting and game progress to the cloud in real time
- Apply cloud save to local once logged in
- Secondary login confirmation
- Privacy compliant (freedom to delete account anytime)

b) Recovery information
- Security questions in case of forgotten User ID and unauthorized profile deletion
- Privacy and usage explanation for user prior to providing said information
- Limited size of data submission to balance safety, privacy, and utility
- Privacy compliant (freedom to delete account anytime)

c) Guest mode
- Offers a mode which collects 0 information, but with content limitations
- Option to create profile with 0 progress loss once inside

d) Moderation
- 2-layer ban system, one from accessing leaderboard & comment, one from the entire program
- Automatic cheat detection and flag based on progress and stats

    4) Miscellaneous

a) Comment and reply tool
- Different comment category
- Spam prevention (character count, message per session)
- Integration with connection diagnostic and ban system
- Developer can reply to comments and can initiate chat
- User can reply to developer reply, detailed chat history

b) Sound & dual language
- Three-level volume settings (off, low, high)
- Sound never overlaps
- English & Chinese localization

c) Customization
- 4 backgrounds
- 2 icon packs
- 2 SFX packs
- 3 button color schemes
- 4 border schemes

d) Extra contents
- Developer journal in Extras
- Update log
- Credits
- Trophy display in tutorial & progress
- Sound deck and Easter eggs

e) Advanced connection diagnostic
- Offline prompt and data loss prevention
- Detect and diagnose China censorship
- Integration with cloud functionality and gameplay

f) Asset pre-load capability

g) Powerful admin tool
- Obfuscated password and access protection
- View and reply to message
- View and action for player profile
- Debug cloud functionality
- View game code in-game
- Detailed action log
